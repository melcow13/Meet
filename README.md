# Meet

Meet is an application for users to find events based on location. It is a serverless progressive web application (PWA) using react and built by using the test-driven development(TDD) technique. 

# Key Features: 
<li>Filter events by city</li>
<li>Show/hide event details.</li>
<li>Specify number of events.</li>
<li>Use the app when offline.</li>
<li>Add an app shortcut to the home screen.</li>
<li>View a chart showing the number of upcoming events by city.</li>
