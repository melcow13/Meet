# Meet

Meet is an application for users to find events based on location. It is a serverless progressive web application (PWA) using react and built by using the test-driven development(TDD) technique. 

#Key Features: 
-Filter events by city.
-Show/hide event details.
-Specify number of events.
-Use the app when offline.
-Add an app shortcut to the home screen.
-View a chart showing the number of upcoming events by city.
